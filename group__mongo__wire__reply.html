Detailed Description

Enumeration Type Documentation

anonymous enum

Flags the server can set in replies.

Enumerator
MONGO_REPLY_FLAG_NO_CURSOR 

Set when get_more is called but the cursor id is invalid.

MONGO_REPLY_FLAG_QUERY_FAIL 

Set when the query failed.

MONGO_REPLY_FLAG_AWAITCAPABLE 

Set when the server suppots the AwaitData query option.

If not set, the client should sleep a little between get_more calls on a tailable cursor. On Mongo >= 1.6, this flag is always set.
